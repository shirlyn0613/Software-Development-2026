# Experiment 4 - Flood Inundation Analysis

This project implements the DEM-based flood inundation lab from `Experiment4_Flood_Inundation.docx`.

## What it does

- generates a synthetic 100x100 DEM
- calculates flooded cells, inundation depth, and flooded percentage
- renders flood extent maps at 40 m and 50 m
- plots water level vs. flooded percentage
- validates physical correctness
- includes optional extensions for routing, barriers, animation, and flood volume

## Main files

- `flood_inundation.py` - main implementation
- `dem_data.npy` - synthetic DEM generated by the script
- `flood_extent_40m.png` - flood visualization at 40 m
- `flood_extent_50m.png` - flood visualization at 50 m
- `flood_curve.png` - flooded percentage curve
- `prompt_log.md` - AI interaction log

## Extension files

- `dem_data_extended.npy` - DEM with building barriers
- `flood_rising.gif` - rising water animation
- `flood_volume_45m.json` - flood volume summary

## Run

```bash
python flood_inundation.py
```

## Notes

- The script uses `matplotlib`, `numpy`, and `Pillow`.
- If `dem_data.npy` already exists and is valid, it will be reused.
- The extension part uses a simple routing model and barrier blocks for demonstration.
