# Alert Log Format
# [YYYY-MM-DD HH:MM:SS] | City: {city} | Rainfall: {rainfall} mm/h | Level: Red
# [YYYY-MM-DD HH:MM:SS] [DEBUG] | City: {city} | Rainfall: {rainfall} mm/h | Level: Red
#
# Notes:
# - Yellow and Red alerts are appended here.
# - Green/Normal readings are not logged.
# - [DEBUG] marks test-generated alert records and should not be treated as real events.
[2026-07-07 12:21:28] [DEBUG] | City: DebugTestCity | Rainfall: 28.8 mm/h | Level: Red
